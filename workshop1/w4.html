IS CODE LANGUAGE?

Free-write on coding: What happens when we swap languaging with coding; and language with code? How would the meaning of this statement change? What would the word code come to mean?

"It’s hard to answer that question when coding is considered a language in itself. It’s a language that is understood by humans and an inanimate object which is fascinating--we’ve never been able to communicated via a language as effectively with actual animate creatures. Yet, we can create pieces of technology that do incredible things using code. The meaning of the statement changes into one of becoming a mediator between all things, not just between peers. Suddenly we can command an object to do complex procedures through a language. Code would just become a different language name with many different dialects within it. Code can also be interpreted to be a set of rules, which is interesting because Wittgenstein discusses how demands are the more primitive form of language. The code language between humans and computers is of the most primitive form. When he describes Person A demanding a stone, and Person B can understand and comply--but without response--it becomes the most simple form of communication. In the same way, we simply order our computers to deliver and they do. So now that I think about it, the definition of “code” is effectively the same: it is a language that delivers a set of rules or demands.
